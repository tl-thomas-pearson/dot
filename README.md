## Install 
```
git clone https://github.com/xsv24/dot.git ~/.dot && ~/.dot/install.sh
```

## Set up


### Install neovim packages
> Start neovim & run :PlugInstall as a vim command.

### Set commit editor to neovim
```
git config --global core.editor "vi" 
```
